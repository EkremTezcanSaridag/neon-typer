# 🎹 Neon Typer

**Neon Typer**, HTML, CSS ve Saf (Vanilla) JavaScript kullanılarak geliştirilen, refleks ve klavye hızı odaklı bir tarayıcı oyunudur. Herhangi bir framework kullanılmadan, tamamen DOM manipülasyonu mantığını kavramak amacıyla geliştirilmektedir.

---

## 🚀 Mevcut Özellikler

- **💣 Bomba Kelime Sistemi:** %20 ihtimalle ekrana bomba kelime düşer. Yazarsan -1 can ve -20 puan cezası alırsın.
- **🔥 Kombo Sistemi:** Üst üste kelime vuruldukça kombo artar. x3 komboda 1.5x, x5 komboda 2x puan çarpanı devreye girer.
- **⚙️ Ayarlar Menüsü:** Zorluk seçimi (Kolay / Orta / Zor) ve ses açma/kapatma özelliği.
- **💥 Juice Efektleri:** Kelime vurulunca parçacık patlaması, can düşünce ekran sarsıntısı.
- **☠️ Game Over Ekranı:** Oyun bitince şık bir ekran açılır, final skoru gösterilir.
- **❤️ Can Sistemi:** 3 canla başlanır, kaçırılan kelimeler can götürür.
- **📈 Dinamik Zorluk:** Skor arttıkça kelimeler hızlanır.

---

## 🎮 Nasıl Oynanır?

1. **OYUNA BAŞLA** butonuna bas.
2. Ekranda yukarıdan aşağıya düşen kelimeleri gör.
3. Kelimeyi klavyeyle yaz ve Enter'a basma, otomatik algılanır.
4. 💣 **Bomba kelimeleri yazmaktan kaçın!**
5. Kombo yaparak puan çarpanını artır.
6. 3 can biterse oyun sona erer.

## 🛠️ Teknolojiler

Bu projede **hiçbir kütüphane veya framework (React, Vue, jQuery vb.) kullanılmamıştır.**

- **HTML5:** Semantik yapı.
- **CSS3:** Flexbox, Animasyonlar, Neon Glow efektleri.
- **Vanilla JavaScript (ES6+):** DOM Manipülasyonu, Event Listener, setInterval, CSS Değişkenleri.
