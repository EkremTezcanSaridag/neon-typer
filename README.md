# 🎹 Neon Typer

**Neon Typer**, HTML, CSS ve Vanilla JavaScript kullanılarak geliştirilen, refleks ve klavye hızı odaklı bir tarayıcı oyunudur. Herhangi bir framework kullanılmadan, tamamen DOM manipülasyonu mantığını kavramak amacıyla geliştirilmektedir.

## 🚀 Mevcut Özellikler

- **💣 Bomba Kelime Sistemi:** %20 ihtimalle ekrana bomba kelime düşer. Yazarsan -1 can ve -20 puan cezası alırsın.
- **⭐ Altın Kelime Sistemi:** %10 ihtimalle ekrana altın kelime düşer. Yazarsan +1 can kazanırsın, can doluysa +50 puan verir.
- **🔥 Kombo Sistemi:** Üst üste kelime vuruldukça kombo artar. x3 komboda 1.5x, x5 komboda 2x puan çarpanı devreye girer.
- **📈 Dinamik Zorluk:** Her 50 puanda kelimeler hızlanır ve daha sık gelir.
- **⚙️ Ayarlar Menüsü:** Ses açma/kapatma özelliği.
- **💥 Juice Efektleri:** Kelime vurulunca parçacık patlaması, can düşünce ekran sarsıntısı.
- **☠️ Game Over Ekranı:** Oyun bitince final skoru gösterilir.
- **❤️ Can Sistemi:** 3 canla başlanır, maksimum 5 cana çıkılabilir.
- **⌨️ Akıllı Input:** Yazdığın kelime hala ekrandaysa input temizlenmez.

## 🎮 Nasıl Oynanır?

1. **OYUNA BAŞLA** butonuna bas.
2. Ekranda yukarıdan aşağıya düşen kelimeleri gör.
3. Kelimeyi klavyeyle yaz, otomatik algılanır.
4. 💣 **Bomba kelimeleri yazmaktan kaçın!**
5. ⭐ **Altın kelimeleri vurarak can kazan!**
6. Kombo yaparak puan çarpanını artır.
7. 3 can biterse oyun sona erer.

## 🛠️ Teknolojiler

Bu projede **hiçbir kütüphane veya framework kullanılmamıştır.**

- **HTML5:** Semantik yapı.
- **CSS3:** Flexbox, Animasyonlar, Neon Glow efektleri.
- **Vanilla JavaScript (ES6+):** DOM Manipülasyonu, Event Listener, setInterval, CSS Değişkenleri.
