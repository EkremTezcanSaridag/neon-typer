# 🎹 Neon Typer

**Neon Typer**, HTML, CSS ve Saf (Vanilla) JavaScript kullanılarak geliştirilen, refleks ve ritim odaklı bir klavye hız oyunudur. Herhangi bir Framework kullanılmadan, tamamen DOM manipülasyonu mantığını kavramak amacıyla geliştirilmektedir.

![Status](https://img.shields.io/badge/Status-In%20Development-yellow)
![Tech](https://img.shields.io/badge/Tech-HTML%20%7C%20CSS%20%7C%20JS-blue)

## 🚀 Özellikler (Planlanan)
- **Kelime Yağmuru:** Yukarıdan aşağıya düşen kelimeleri zamanında yakalama.
- **Dinamik Zorluk:** Zamanla artan düşüş hızı ve spawn süresi.
- **Görsel Estetik:** Cyberpunk / Retro Terminal arayüzü.
- **Skor ve Can Sistemi:** Hatasız yazım serileri ve hata cezaları.

## 🛠️ Teknolojiler
Bu projede **hiçbir kütüphane veya framework (React, Vue, jQuery vb.) kullanılmamıştır.**
* **HTML5:** Semantik yapı.
* **CSS3:** Flexbox, Grid ve Animasyonlar.
* **Vanilla JavaScript (ES6+):** DOM Manipülasyonu, Event Listener, Asenkron Fonksiyonlar.
